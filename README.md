# primo

## primo-generator-mock-test
单元测试代码Mack自动生成插件  
（原项目名：auto-generate-test-maven-plugin）

自动生成mock测试代码插件,通过Maven插件配置，一键生成配置包/类的单元测试代码，极大减少开发写单元测试代码的时间，提高单元测试覆盖率  

### 大概功能
- 单元测试代码的自动生成
- 分支覆盖代码判断与生成 
- 构造实体的参数自动构建

## primo-generator-mybatis-plus 
项目增删改查非业务代码全自动生成插件  

自动生成增删改查代码插件,通过Maven插件配置，一键生成单表的所有CRUD操作

### 大概功能  
- 单表增删改查
- 单表的条件查询，分页查询
- 基于druid，数据库监控
- 统一查询条件模型
- 统一controller层模型
